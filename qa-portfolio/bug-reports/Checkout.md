# [Checkout] Order confirmation page not displayed after successful payment

## Environment
- Environment: Staging
- Browser: Chrome 121
- OS: Windows 11
*Please see environment file for more info*

## Steps to Reproduce
1. Add a product to the cart
2. Proceed to checkout
3. Complete payment using credit card
4. Observe the redirect after payment

## Expected Result
User is redirected to the order confirmation page.

## Actual Result
User remains on the checkout page without any confirmation message.

## Severity
Major

## Priority
High

## Additional Notes
- Issue occurs intermittently
- No console error observed
- Possible backend timeout
